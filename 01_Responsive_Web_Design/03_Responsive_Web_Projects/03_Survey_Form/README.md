# AI Developer Community Survey

A responsive survey form designed to collect insights from developers about their programming journey, technology interests, and AI adoption.

This project is part of my FreeCodeCamp Responsive Web Design Certification journey.

---

## 🚀 Project Overview

The AI Developer Community Survey is a modern responsive form that allows developers to share:

- Personal information
- Development experience
- Programming interests
- AI/ML interests
- Learning goals
- Community feedback

The goal is to understand how developers learn, build, and use modern technologies.
