# Tribute Page - Alan Turing

A responsive tribute webpage dedicated to Alan Turing, one of the pioneers of computer science, artificial intelligence, and modern computing.

This project is part of my FreeCodeCamp Responsive Web Design Certification journey.

---

## 🚀 Project Overview

This tribute page presents the life, achievements, and impact of Alan Turing.

The page includes:

- Hero introduction
- Biography section
- Major contributions
- Timeline of achievements
- Famous quotes
- Legacy section
