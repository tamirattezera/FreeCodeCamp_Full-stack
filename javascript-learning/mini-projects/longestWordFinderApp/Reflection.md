# Reflection

## What Problem Did I Solve?

I built a function that finds and returns the length of the longest word in a sentence.

## What Concepts Did I Practice?

- Strings
- Arrays
- Loops
- Conditionals
- Variables
- Problem decomposition

## What Pattern Did I Learn?

Traverse -> Compare -> Update -> Return

Steps:

1. Traverse every item.
2. Compare the current item with the best result found so far.
3. Update the result if a better value is found.
4. Return the final result.

## Where Can This Pattern Be Used?

- Finding the highest score
- Finding the largest transaction
- Finding the most expensive product
- Finding the longest name
- Finding the highest-rated review
- AI confidence ranking systems

## What Was Challenging?

Understanding how to keep track of the current maximum value while iterating through the array.


## What Did I Learn?

A complex problem can be broken into smaller steps:

- Split
- Loop
- Compare
- Update
- Return

This makes problem-solving easier and more structured.

## Can I Rebuild This Without Looking?

Yes.

I understand:

- Why split() is used.
- Why a loop is needed.
- Why a variable stores the current maximum length.
- How the comparison logic works.
