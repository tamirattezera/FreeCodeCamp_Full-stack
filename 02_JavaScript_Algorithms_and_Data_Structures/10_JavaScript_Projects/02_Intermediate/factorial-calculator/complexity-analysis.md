# Complexity Analysis

## Time Complexity

O(n)

Reason:

The loop runs once for every number from 1 to n.

Example:

If n = 7, the loop executes 7 times.

If n = 100, the loop executes 100 times.

## Space Complexity

O(1)

Reason:

Only a few variables are used regardless of the size of the input.

Variables:

* num
* result
* i

No additional arrays or objects are created.
