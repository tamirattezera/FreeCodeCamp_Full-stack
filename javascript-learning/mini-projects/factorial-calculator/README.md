# Factorial Calculator

## Problem Statement

Create a program that calculates the factorial of a non-negative integer.

A factorial is the product of an integer and all positive integers below it.

Examples:

- 5! = 5 × 4 × 3 × 2 × 1 = 120
- 3! = 3 × 2 × 1 = 6
- 0! = 1

## Input

A non-negative integer.

Example:

7

## Output

The factorial of the given number.

Example:

5040
