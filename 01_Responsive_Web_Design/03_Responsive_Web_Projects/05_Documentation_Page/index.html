Introduction

JavaScript is a programming language used to create interactive and dynamic web applications.

It runs inside browsers and also powers backend systems through platforms like Node.js.

console.log("Hello JavaScript");
Variables

Variables store information that can be reused in programs.

let username = "Tamirat"; const age = 25;
  • let allows reassignment
  • const creates immutable references
Data Types

JavaScript supports different types of values.

String Number Boolean Object Array
Functions

Functions are reusable blocks of code.

function greet(){ return "Hello Developer"; }
Arrays

Arrays store multiple values in a single variable.

const skills = [ "HTML", "CSS", "JavaScript" ];
Objects

Objects store data using key-value pairs.

const developer = { name:"Tamirat", role:"Full Stack Developer" };
DOM Manipulation

The DOM allows JavaScript to change HTML content dynamically.

document.querySelector("h1");
